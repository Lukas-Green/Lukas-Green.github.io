01

Principles.

Five rules that hold whenever a decision is hard. If a proposal breaks one, rewrite the proposal.

01

No gradients.

Flat color only. Depth comes from hairline rules, tabular alignment, and a 3% paper grain — never a color ramp.

02

One accent, used sparingly.

Ink‑teal marks interaction, the one key metric, and a single hero moment per page. If it's decorative, remove it.

03

Type does the heavy lifting.

Fraunces for authority, Inter for clarity, JetBrains Mono for numbers. Tabular figures on by default.

04

Structured whitespace.

8‑pt scale, wide gutters, generous vertical rhythm. Expensive feels empty until you stop to read it.

05

Motion is quiet.

200ms ease‑out, 1–2px translate on hover, underline draw on links. No parallax, no scroll‑jacking, no orbs.

06

Two brands, one DNA.

Upskilled on warm paper; Nick on cool paper. Everything else — type, scale, accent, components — identical.

02

Color.

Two papers, four inks, one accent. That's the entire palette. Avoid introducing a second hue — restraint is the point.

Canvas

Paper · warm #F2ECE0 Upskilled Consulting canvas. Institutional.
Paper · cool #FAFAF6 nickmccarty.me canvas. Quieter.
Rule #D9D3C7 Hairlines, card borders.
Rule · soft #E4DFD3 Hover states, subtle fills.

Ink & accent

Ink #12110F Body copy, headlines.
Ink · dim #5A564E Secondary text, captions.
Ink · faint #8A857A Timestamps, meta. Non‑essential only.
Accent #0E4F4C Ink‑teal. Interaction & one hero moment.
Contrast rules. Ink on either paper clears 14:1 (AAA). Ink‑dim on warm paper is 5.2:1 (AA body). Ink‑faint is 3.1:1 — large text or non‑essential meta only, never body copy. Accent on either paper is 8.9:1 — safe for links, buttons, and running numerics.

06

Motion.

Three named interactions. Everything else is still. Duration 200ms, ease cubic-bezier(.2,.6,.2,1). No parallax, no scroll‑jacking, no hero animations.

01 · Lift

Card hover.

1–2px translate up, border darkens from rule to ink. Signals "clickable" without shadow.

Hover me
02 · Tint

Primary button.

Fill darkens one step; arrow translates 3px right. Confidence, not bounce.

Hover me
03 · Underline draw

Inline link.

Accent underline draws left‑to‑right under the word. Color shifts to accent.

Hover me
Reduced motion. All transitions collapse to 0.01ms under prefers-reduced-motion. Already wired into system.css. Do not override.
07

Voice.

Write like Nick actually talks about his work: specific, direct, quietly confident, occasionally dry. The work is the proof — the copy doesn't have to oversell.

Do

Geospatial AI, built to ship.
A short, honest cycle that proves or kills the approach.
Data teams that don't need us next quarter.
Tiled inference is the boring part — and the part that breaks first.
We aim to be missed, not necessary.

Don't

Leveraging cutting‑edge AI to unlock synergies.
Let's build something amazing together!
Passionate about data. Obsessed with impact.
Revolutionizing how enterprises think about ML.
Your trusted partner in the AI journey.
Three rules. (1) Prefer a specific noun to an abstract one — "drone orthomosaic," not "geospatial data asset." (2) State the outcome before the method. (3) If a sentence could appear on any consulting site, rewrite it. The voice is the moat.
08

Implementation.

Everything any dev needs to ship the system in thirty seconds. Copy the font link, copy the token block, start building.

1 · Google Fonts

Drop into <head>. Variable Fraunces for opticals; Inter and JetBrains Mono at the weights we actually use.

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Fraunces:opsz,wght@9..144,400;9..144,500;9..144,600&family=Inter:wght@400;500;600&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">

2 · Design tokens

Drop into your root stylesheet. Swap --paper per brand — nothing else forks.

:root { /* Canvas — override --paper per brand */ --paper-warm: #F2ECE0; /* Upskilled */ --paper-cool: #FAFAF6; /* Nick personal */ --paper: var(--paper-warm); /* Ink */ --ink: #12110F; --ink-dim: #5A564E; --ink-faint: #8A857A; --rule: #D9D3C7; /* Accent */ --accent: #0E4F4C; --accent-ink: #F2ECE0; /* Type */ --font-display: 'Fraunces', Georgia, serif; --font-body: 'Inter', -apple-system, sans-serif; --font-mono: 'JetBrains Mono', ui-monospace, monospace; /* Scale (8pt) · motion */ --s-1: 4px; --s-2: 8px; --s-3: 12px; --s-4: 16px; --s-5: 24px; --s-6: 32px; --s-7: 48px; --s-8: 64px; --s-9: 96px; --s-10: 128px; --ease: cubic-bezier(.2,.6,.2,1); --dur: 200ms; }

3 · Per‑brand override

One rule at the body level per brand. That's the entire brand fork.

/* Upskilled Consulting */ body.brand-upskilled { --paper: var(--paper-warm); } /* nickmccarty.me */ body.brand-nick { --paper: var(--paper-cool); }
The full stylesheet. system.css ships the reset, nav, buttons, list rows, service rows, metrics, pull quotes, footer, responsive breakpoints, and reduced‑motion handling. ~15KB. Link it and the rest of the system is yours.
